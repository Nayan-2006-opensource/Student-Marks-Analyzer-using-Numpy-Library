# Student Marks Analyzer using NumPy

A Python project that analyzes student marks using the NumPy library. The project demonstrates how NumPy can be used to perform fast numerical computations and statistical analysis on student performance data.

## Features

- Calculate total marks of each student
- Calculate average and percentage
- Find the class topper and lowest scorer
- Calculate class average
- Subject-wise highest, lowest, and average marks
- Identify each student's best and weakest subject
- Pass/Fail analysis
- Student ranking
- Display the top 3 and bottom 3 students
- Median and standard deviation for each subject

## Technologies Used

- Python 3
- NumPy

## Dataset

The project generates marks for:

- 15 students
- 5 subjects
- Marks ranging from 0 to 100

The marks are stored in a **15 × 5 NumPy array**, where each row represents a student and each column represents a subject.

## Concepts Practiced

- NumPy arrays
- Indexing and slicing
- Aggregation functions
- Axis operations
- Boolean masking
- Statistical analysis
- Ranking using NumPy
- Vectorized computations

## How to Run

1. Clone the repository.

```bash
git clone https://github.com/your-username/Student-Marks-Analyzer.git
```

2. Install NumPy.

```bash
pip install numpy
```

3. Run the program.

```bash
python marks_analyzer.py
```

## Learning Objective

The objective of this project is to strengthen Python and NumPy fundamentals by building a real-world data analysis application using vectorized operations instead of manual calculations wherever possible.
